# FanCardShop â€” Final Embedded Edition (Full-resolution)

This repository is a demo-ready static site for FanCardShop. After running the included download script, the site will contain full-resolution celebrity images embedded locally and a created 1920x600 header banner.

## Quick start
1. Unzip this package.
2. Run `download_images.sh` (macOS/Linux/WSL) or `download_images.ps1` (Windows PowerShell).
3. Confirm `assets/images/celebrities/` contains the 12 full-res images and `assets/images/banner/banner.jpg` exists.
4. Push to GitHub and enable Pages.
